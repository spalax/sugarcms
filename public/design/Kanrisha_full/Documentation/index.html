Introduction

Kanrisha is a premium HTML5 responsive admin template, with a sweet clean and minimal overview, suitable for iPad and iPhone resolutions, it's coming with +14 plugin ( Some of them are customized for the need of kanrisha ), and 6 awesome patterns.

Kanrisha is a responsive template with two column, the sidebar column use the concept of the Tabs, this concept give Kanrisha a good look on the tablets devices, It's Cool Modern and Elegant as well :)

File Structure

Kanrisha file structure are well organized on 5 Folders:

  • CSS: Contain the main css style and the styles files of some plugins
  • HTML: Contain the HTML files
  • Javascript: Contain the Kanrisha main script and the plugins
  • Images: Contain the icons, patterns, logo, and some image related with the plugins.
  • Design: Contain the main PSD file.

HTML Structure

Kanrisha is a responsive template with 1000px grid with 16 columns, 4 are for the sidebar and 12  for contents, It use the media queries to fit the different devices resolutions.

Grids

The Contents divided to 12 Columns, above on the HTML Structure section, and when I make:

{ Here The Grid Blocks }

You can put your blocks like this:

<div class="g_XX">

On the place of g_XX you can chose any number from 2, 3, 4, 6, 8, 9, 10, 12, for Ex:

<div class="g_3">
    { Contents Go Here }
</div>
<div class="g_9">
    { Contents Go Here }
</div>

Icons

Let say you want to put a custom icon in the widget header, all you need is to add a class on the widget header div like this, i_SIZE_NAME, SIZE is the size of the icon (16, 32 ..), the NAME is the name of the icon of course.

See this example:

<div class="widget_header">
    <h4 class="widget_header_title wwIcon i_16_chats">Chats</h4>
</div>

Here we have the class i_16_chats, mean an icon with the name chats and size 16px, Okay what happen if I want to add a new icon?

Easy go to /Images/Icons/<YOUR ICON SIZE> and paste your icon and give it a name the same as the class, and add this snippet on the Main CSS file in the Icons section:

.i_SIZE_NAME {
  background: url("../Images/Icons/SIZE/i_SIZE_NAME.png") no-repeat scroll transparent;
}

So for example, if we have an icon with size 16px and name undo, we add it like this:

.i_16_undo {
  background: url("../Images/Icons/16/i_16_undo.png") no-repeat scroll transparent;
}

and we use it like that:

<div class="widget_header">
    <h4 class="widget_header_title wwIcon i_16_undo">Undo</h4>
</div>

This awesome mechanism work on the Menu, The Big Top Buttons, The Sidebar, Top Panel and Buttons ..
